# 自然法則整合性チェックリスト

自然法則整合性とは、AI支援による行動が、生命と文明を成立させる物質的・生態的・エネルギー的・社会的・時間的制約を尊重しているかを評価することです。

これは宗教的主張ではありません。  
システム安全のためのチェックリストです。

---

## チェックリスト

| 領域 | 確認事項 |
|---|---|
| エネルギー | エネルギーコストと廃熱を考慮しているか |
| 物質 | 持続不能な採掘や脆弱な供給網に依存していないか |
| 水 | 水の利用可能性と水質を守るか、劣化させるか |
| 土壌と生物 | 生きたシステムを支えるか、損なうか |
| 気候レジリエンス | 暑熱、干ばつ、洪水、嵐への回復力を高めるか |
| 廃棄物 | 不可逆な廃棄物を生むか、循環回収を可能にするか |
| 時間軸 | 短期利益のために長期負担を作っていないか |
| 人間社会 | 尊厳、監督、社会的信頼を保つか |
| 失敗モード | システムが失敗・悪用・過剰拡大した時に何が起きるか |
| 可逆性 | 停止、修理、監査、巻き戻しが可能か |

---

## 最低基準

短期的成功のために、長期的な生命維持システムの基盤を劣化させる行動は、自然法則整合性を満たしません。
